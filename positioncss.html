position: static;

Position static digunakan untuk mengatur element menjadi static secara default. Element aka mengikuti pos normal secure default, element tidak dipengaruhi oleh property seperti top, bottom, left down right.:

Elemen ini datur menggunakan post: static;




position: relative;

Sebuah element HTML yang menggunakan position relative akon terletak pad pos normal. Mengatur property atas, kanan, Konten lain tian aka disesuaikan agar sesuai dengan celah yang ditinggalkan oleh element tersebut.:

Elemen ini diatur menggunakan posisi : relative;



position: fixed;

Sebuah element HTML yang di setting dengan position fixed akan memiliki sifat tetap, tanpa ada perubahan bahkan jika halaman website di scroll. Berlaku pengaturan left, bottom, top dan right page position fixed:

Elemen ini diatur menggunakan posisi: fixed;















position: absolute;

Element HTML yang menggunakan position absolute akan diposisikan relatif dengan element lain yang mendahului nya yang terdekat, bukan relatif terhadap layar secara normal.:

Elemen ini diatur menggunakan posisi: relative;
Elemen ini diatur menggunakan posisi: absolute;
